1. Name of Repository: Code

2. Description: Contains all the code relating to LogiSketch

3. Sub-directories:

	FrontEnd
	    - Contains code relating to UI (windows, panels)
	Logo	
	    - image files for logo icon
	Recognition
	    - code relating to recognition algorithms
	Util
	    - libraries, test rigs/frameworks, file type converters, and other 
              utility classes

4. Commonly Found File Types in the Sub-Folders (extensions):

	C# source files (.cs, .csproj) 
	Image files (.bmp, .ico)
	XML file types (.resx, .xaml)
	Text Files (.txt)
	Microsoft Visual Studio Specific Files (.sln)
	Dynamically Linked Libraries (.dll)
